Automatização da limpeza de dados do e-SISTAFE

Usando R para automatizar a compilação e limpeza de ficheiros exportados e-SISTAFE

Introdução

Este documento foi elaborado para a equipa da DAF MEC com o objectivo de demonstrar como é possível automatizar a compilação e limpeza dos dados exportados do sistema e-SISTAFE utilizando a linguagem de programação R.

Actualmente, este processo é frequentemente realizado de forma manual copiando e colando dados entre ficheiros Excel, aplicando filtros à mão, e repetindo os mesmos passos a cada novo relatório. Este tipo de trabalho consome tempo, é propenso a erros, e dificulta a rastreabilidade das decisões tomadas durante a limpeza dos dados.

Com a abordagem apresentada aqui, os mesmos passos de limpeza e organização são executados automaticamente pelo R, de forma consistente e reproduzível — independentemente do número de ficheiros ou do período analisado. O resultado é um conjunto de dados limpo, filtrado e pronto para análise, gerado em segundos.

Este documento está organizado da seguinte forma:

  1. Configuração do ambiente de trabalho
  2. Detecção automática dos ficheiros e-SISTAFE
  3. Extracção de metadados
  4. Carregamento e limpeza dos dados
  5. Classificação e categorização das variáveis-chave

Extrair metadados dos ficheiros

Os nomes dos ficheiros eSISTAFE contêm informações úteis (como datas e códigos). A função extrair_meta_extracto é uma função personalizada desenvolvida especificamente para este projecto e definida no ficheiro Scripts/functions.R. O seu objectivo é extrair automaticamente informações de identificação — os chamados metadados — a partir do nome e do conteúdo de cada ficheiro e-SISTAFE exportado.

Concretamente, a função identifica e extrai elementos como o período de reporte (mês e ano), o código UGB, e outros identificadores presentes no nome do ficheiro ou no cabeçalho do extracto. Estes metadados são essenciais para garantir que, ao combinar múltiplos ficheiros numa única tabela, cada linha de dados possa ser correctamente associada à sua fonte original.

Em vez de inserir estas informações manualmente para cada ficheiro — o que seria moroso e sujeito a erros — a função faz este trabalho de forma automática e consistente para todos os ficheiros detectados.

df_metadados <- map(paths, extrair_meta_extracto) |> 
                 list_rbind(names_to = "arquivo_fonte")

Carregar e limpar ficheiros no R

Carregar ficheiros

Agora carregamos todos os ficheiros eSISTAFE e combinamo-los num único conjunto de dados. A funcção map aplica a mesma operação a cada ficheiro, e list_rbind junta tudo numa única tabela.

df <- map(paths, ~read_excel(.x, col_types = "text")) |> 
  set_names(basename(paths)) |> 
  list_rbind(names_to = "file_name")

Extrair código UGB

Neste passo, criamos uma nova variável chamada ugb_id, extraindo os primeiros 9 caracteres da coluna ugb. Estes 9 caracteres representam o código único que identifica cada UGB no sistema e-SISTAFE, e será esta variável — ugb_id — que usaremos posteriormente para cruzar os dados com a nossa tabela de referência das UGBs do sector da educação.

Aproveitamos também este passo para converter todas as colunas financeiras (de dotacao_inicial até liq_ad_fundos_via_directa_lafvd) de texto para valores numéricos, o que é necessário para realizar quaisquer cálculos matemáticos mais tarde.

Categorizar CED por nível

Aqui criamos uma nova variável que identifica o nível de reporte do orçamento/execução dentro do conjunto de dados. Isto é importante porque os dados eSISTAFE contêm valores em diferentes níveis de agregação do CED. Com base na estrutura do código CED , classificamos cada linha em grupos (A, B, C, D) — onde “A” representa o nível mais detalhado e “D” o mais agregado.

df_limpeza_5 <- df_limpeza_4 |> 
    dplyr::mutate(
      ced_blank_class = ifelse(!is.na(ced), "Keep", "Remove"),
      ced_group = dplyr::case_when(
        !stringr::str_ends(ced, "00")                                           ~ "A",
        stringr::str_ends(ced, "00") & !stringr::str_ends(ced, "000") & !stringr::str_ends(ced, "0000") ~ "B",
        stringr::str_ends(ced, "000") & !stringr::str_ends(ced, "0000")        ~ "C",
        stringr::str_ends(ced, "0000")                                         ~ "D",
        TRUE                                                                   ~ NA_character_
      )
    ) |> 
      select(-ced_blank_class) |> 
      relocate(ced_group, .after = ced)
