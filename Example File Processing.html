Exemplo de processamento de ficheiro eSISTAFE

Configuração

Primeiro, vamos começar carregando o código de programação necessário para processar o nosso ficheiro e-SISTAFE. No R, isso é feito através das funções ‘library()’ e ‘source()’.

library(tidyverse)
library(readxl)
library(janitor)
source("Scripts/functions.R")

Carregamento

df <- map(paths, ~read_excel(.x, col_types = "text")) |> 
  set_names(basename(paths)) |> 
  list_rbind(names_to = "file_name")
